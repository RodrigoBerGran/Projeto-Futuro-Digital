<===================== Conceitos Aplicados: =====================>

    • useState e useEffect

    • Renderização condicional

    • Manipulação de estados complexos

    • Consumo de API REST simulada

    • CRUD completo

    • Componentização

    • Controle de fluxo com modais

    • Organização de layout com Flexbox/Grid

    • Implementação de rotas utilizando React Router DOM

    • Navegação entre páginas sem recarregamento

    • Estruturação da aplicação como SPA (Single Page Application)

    • Uso de <Link> para navegação interna

<===================== Funcionalidades: =====================>

    • Autenticação
    - Login com validação de credenciais
    - Mensagens de erro para usuário ou senha inválidos

    • Cadastro de Usuário
    - Criação de novo usuário
    - Validação de campos obrigatórios
    - Persistência no mock-end

    • Gestão de Clientes (CRUD Completo)
    - Criar novo cliente
    - Buscar cliente por nome
    - Editar cliente com modal de confirmação
    - Excluir cliente com confirmação
    - Atualização dinâmica da interface
    - Persistência de dados utilizando API simulada

    • Sistema de Busca
    - Busca acionada por botão ou tecla Enter
    - Exibição condicional de resultados
    - Mensagem para nenhum cliente encontrado

    • Classificação Inteligente
    - Etiqueta automática baseada no consumo:
        > Baixo
        > Médio
        > Alto

    • Experiência do Usuário
    - Modais de confirmação
    - Feedback visual de sucesso e erro
    - Interface organizada em cards
    - Layout estruturado e padronizado
